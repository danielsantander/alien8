# MySQLScan.sh
Performs `nmap` scan to find hosts with open port 3306.

Usage: `./MySQLScan.sh`

Output Files:
- ResultsMySQLScan
- ResultsOpenMySQLPorts

# MySQLScan2.sh
Advanced version of `MySQLScan.sh` allowing user input.

Usage: `./MySQLScan2.sh`

Input when prompted:
- first ip address
- last octet of the last IP address
- port number
